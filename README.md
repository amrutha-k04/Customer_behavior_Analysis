# Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from data collection to business insights. The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard creation, report generation, and presentation preparation.

---

## Dataset

The dataset contains business-related data used for analysis and visualization.

The dataset was analyzed to:

* Understand the data structure
* Identify missing values and duplicates
* Clean and prepare the data
* Generate meaningful business insights

## Project Steps

### 1. Data Loading

* Imported the dataset into Python.
* Loaded the data using Pandas.

### 2. Exploratory Data Analysis (EDA)

* Checked data types and summary statistics.
* Identified missing values.
* Analyzed distributions and relationships.
* Created visualizations for better understanding.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected data types.
* Prepared the dataset for analysis.

### 4. SQL Analysis

Executed SQL queries using:

* PostgreSQL
* MySQL
* SQL Server

Performed:

* Filtering
* Sorting
* Aggregation
* Joins
* Group By
* Subqueries
* Business KPI calculations

### 5. Power BI Dashboard

Created an interactive dashboard featuring:

* KPI Cards
* Bar Charts
* Line Charts
* Pie Charts
* Slicers and Filters
* Business Performance Insights

### 6. Report Creation

Prepared a detailed report summarizing:

* Data cleaning process
* Key findings
* Business insights
* Recommendations

### 7. Presentation

Created a professional project presentation using **Gamma AI** to communicate the analysis and insights effectively.
